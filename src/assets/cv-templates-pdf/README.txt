CV TEMPLATES PDF FOLDER

This folder is for storing PDF template images for the different CV styles:

1. professional.pdf - For the Professional template style
2. creative.pdf - For the Creative template style
3. executive.pdf - For the Executive template style
4. academic.pdf - For the Academic template style

After adding your PDF files to this folder, the application will automatically use them as template previews on the Templates page.

IMPORTANT: Make sure to name the files exactly as listed above for them to be properly recognized by the application.
